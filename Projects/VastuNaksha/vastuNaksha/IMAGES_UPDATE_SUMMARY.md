## üîÑ If You Want to Change Images Later

To update any design image:

1. Go to `/src/data/designsData.js`
2. Find the design you want to update
3. Replace the `image` URL with a new Unsplash URL
4. The change will reflect immediately (Vite hot reload)

### How to Find Images

- Visit [unsplash.com](https://unsplash.com)
- Search for: "modern house", "interior design", "apartment", etc.
- Copy image URL: Right-click ‚Üí Copy image link
- Format the URL: `https://images.unsplash.com/photo-XXXX?w=800&h=600&fit=crop&q=80`
